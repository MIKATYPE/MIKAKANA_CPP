美佳のタイプトレーナ カナ編 Ｖｅｒ２．０３．０１ Windows C++版ソースコード

　教育用のタイプ練習ソフト「美佳のタイプトレーナ カナ編V2.03」のWindows C++版ソースコードです。
　このソースコードの著作権は今村二朗が所有します。
　改変、転載、他ソフトの使用など自由にお使いください。
　
注意事項

１．

Visual Studio 2022
Visual C++ 2022
64ビット版で動作確認済です

２．

Visual C++がインストールされていないパソコンで動作させるにはVisual C++ のランタイムライブラリをインストールする必要があります。

３．

Visual Studioでコンパイルするときにはプロジェクト名をMIKAKANAにして下さい。

４．

このコースコードでは成績ファイル書き込み時にエラーがあってもダイアログを表示しません。

５．

練習テキストを変更するときには練習テキスト文字列の最後にストッパーL""を追加してください。
ただしJAVA版のソースコードではストッパーの""は必要ありません。

例

wstring MIKA_kana_tango_table[] = {
	L"あいくるしい",
	L"あいこくしん",
	L"あいさつ",
	L"あいて",
	L"" 
};

６.

キー入力は半角カナ無変換直接入力のほかに英数モードでも可能です。
